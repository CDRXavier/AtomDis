# AtomDis
Disassembler/Parser for AtomBIOS

Effectively a clone of Mhopf's work at https://cgit.freedesktop.org/~mhopf/AtomDis/

It works! This is very important.

```ansi
xavier@LAPTOP-3010014:/mnt/Volume_A2/rom_parser/mhoph$ ./atomdis
'/media/xavier/THINGIES/capeverdeturks_1.rom' i
Read 20000 bytes of data from /media/xavier/THINGIES/capeverdeturks_1.rom

Table Format Rev.   01
Table Content Rev.  01
BIOS Boot Message:  '
TURKS ProL C33402 GDDR5 64Mx32                                              
'
Subsystem Vendor ID 1028
Subsystem ID        2b20
```
